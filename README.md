# FakeSoraAPI

## Quick Started

1. Clone project

```bash
git clone git@github.com:SoraWebui/FakeSoraAPI.git
```

2. Install dependencies

```bash
cd FakeSoraAPI && yarn
#or
cd FakeSoraAPI && npm install
#or
cd FakeSoraAPI && pnpm install
```

3. Run it

```bash
yarn dev
#or
npm run dev
#or
pnpm dev
```

4. Open [http://localhost](http://localhost) with your browser to see it.
